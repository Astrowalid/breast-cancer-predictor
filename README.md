# Classification du Diagnostic du Cancer du Sein 🎗️

Ce projet, réalisé dans le cadre du module **Intelligence Artificielle** (Filière ADIA - IISE), vise à développer un système capable de prédire si une tumeur mammaire est bénigne ou maligne à partir de mesures cliniques.

## 🔗 Liens du Projet
* **Application Web (Streamlit) :** [[Breast Cancer Prediction Tool](https://breast-cancer-predictor-adia.streamlit.app/)]
* **Dépôt GitHub :** [https://github.com/Astrowalid/breast-cancer-predictor]

## 🚀 Fonctionnalités
* **Exploration interactive :** Visualisation des caractéristiques cliniques (Moyenne, Erreur Standard, Pire).
* **Prédiction en temps réel :** Interface permettant de saisir les mesures d'une biopsie pour obtenir un diagnostic instantané.
* **Comparaison de modèles :** Évaluation des performances via Précision, Rappel et Score F1.

## 🛠️ Installation
Pour lancer l'application localement :
1. Cloner le dépôt : `git clone https://github.com/Astrowalid/breast-cancer-predictor`
2. Installer les dépendances : `pip install -r requirements.txt`
3. Lancer Streamlit : `streamlit run breastCancerApp/app.py`
